# 🌦 AtmosIQ – Smart Weather Intelligence Web App

AtmosIQ is a modern weather intelligence web application that provides real-time weather data, weather analysis, and predictions using the OpenWeather API.  
The application features a beautiful animated UI, glassmorphism design, interactive city map, and downloadable weather reports.

This project demonstrates how weather APIs can be integrated with modern frontend design to create an interactive user experience.

---

## 🚀 Features

• 🌍 Search weather by city name  
• 🌡 Real-time temperature display  
• 💧 Humidity and pressure information  
• 🌬 Wind speed and direction  
• 📊 Weather analysis summary  
• 🔮 Smart weather prediction message  
• 🗺 Interactive city map view  
• 📄 Download weather report as PDF  
• 🎨 Modern animated glassmorphism UI  
• ⚡ Smooth hover and gradient animations  

---

## 🛠 Technologies Used

- **HTML5** – Structure of the web application  
- **CSS3** – Styling, animations, and glassmorphism UI  
- **JavaScript (ES6)** – Weather API integration and logic  
- **OpenWeather API** – Real-time weather data  
- **Google Maps Embed** – Location visualization  
- **jsPDF** – Weather report generation in PDF format
